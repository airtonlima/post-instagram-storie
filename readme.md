
App que fiz para testar o uso da [instagrapi](https://github.com/adw0rd/instagrapi) no meu app em node.js.

---

No node:
Usa a **python-shell** para rodar um programa em python e enviar argumentos;

No python:
Usa a **requests** para fazer requisições (No meu caso baixar a imagem);
Usa a **Pillow** para manipular imagem;
Usa a **instagrapi** para realizar login e postar storie

Obs.:
Existe uma api de instagram em node.js, a [instagram-private-api](https://github.com/dilame/instagram-private-api), porém a mesma não tem a função de postar storie com link e ai o motivo desse teste.
Meu app já rodava em node então só escrevi o programa em python usando a **instagrapi** e usei o **python-shell** pra executar ele do meu programa em node. 

---

É a melhor maneira? Definitivamente não.
Resolveu o meu problema a curto prazo? Com certeza.
